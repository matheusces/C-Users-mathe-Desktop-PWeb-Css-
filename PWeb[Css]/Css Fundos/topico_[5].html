Fundos CSS

As propriedades de plano de fundo do CSS são usadas para definir os efeitos de segundo plano para elementos.

Propriedades de fundo CSS:

  • cor de fundo
  • imagem de fundo
  • fundo de repetição
  • fundo de ligação
  • posição de fundo

Cor de fundo

A background-color propriedade especifica a cor de fundo de um elemento.

A cor de fundo de uma página é configurada assim:

Exemplo

body {
   background-color: lightblue;
}

Com CSS, uma cor é mais frequentemente especificada por:

  • um nome de cor válido - como "vermelho"
  • um valor HEX - como "# ff0000"
  • um valor RGB - como "rgb (255,0,0)"

Veja os valores de cores CSS para obter uma lista completa de possíveis valores de cores.

No exemplo abaixo, os elementos <"h1">, <"p"> e <"div"> têm cores de fundo diferentes:

Exemplo

h1 {
   background-color: green;
}
div {
   background-color: lightblue;
}
p {
   background-color: yellow;
}

Imagem de fundo

A background-image propriedade especifica uma imagem para usar como plano de fundo de um elemento.

Por padrão, a imagem é repetida para cobrir todo o elemento.

A imagem de fundo de uma página pode ser configurada assim:

Exemplo

body {
   background-image: url("paper.gif");
}

Abaixo está um exemplo de uma combinação ruim de texto e imagem de fundo. O texto é dificilmente legível:

body {
   background-image: url("bgdesert.jpg");
}


Nota: Ao usar uma imagem de fundo, use uma imagem que não perturbe o texto.
