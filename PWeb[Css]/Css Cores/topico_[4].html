Cores CSS

As cores são especificadas usando nomes de cores predefinidos ou valores RGB, HEX, HSL, RGBA, HSLA.

Nomes de cores

Em HTML, uma cor pode ser especificada usando um nome de cor:

Tomato

Orange

DodgerBlue

MediumSeaGreen

Gray

SlateBlue

Violet

LightGray

O HTML suporta 140 nomes de cores padrão .

Cor de fundo

Você pode definir a cor de fundo para os elementos HTML:

Hello World

Lorem ipsum...

Exemplo

<"h1 style="background-color:DodgerBlue;">Hello World
<"p style="background-color:Tomato;">Lorem ipsum...
